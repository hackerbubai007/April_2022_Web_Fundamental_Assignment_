List Element
  • It will help us to structure the data in a particular order or in unorder.
  • It is divided into 2-type:
    • Order List
      1. Html
      2. CSS
      3. Javascript
    • Unorder List
      • Html
      • CSS
      • Javascript

Attributes of OL and UL

  • Ordered List
    • Attribute:
      1. type="1","a","A","i","I"
      2. start attribute only work with number for example start="1001"
    • By default number will be printed.
  • Unordered List
    • Attribute: type="disc","circle","square","none"
    • By default disc will be printed.
