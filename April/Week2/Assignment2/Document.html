Structure of Html Document

Purpose of DOCTYPE in the Html Document

  • To specify the version of HTML 5.0
  • If not mention document declaration, then version will be 4 or less then 4 automatically.

  • It is divided into two parts:

  1. Document Declaration
  2. Document Scope

  1. Document Declaration

It will let us know which version of Html we are using:

  • HTML 1.0
  • HTML 2.0
  • HTML 4.0
  • HTML 5.0

  1. Document Scope

Difference between Head section and Body section

  1. Head Section: Loaded in the browser memory

  1. Body Section: Directly loaded in the body part of browser
